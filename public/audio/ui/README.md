# UI Sounds

Settlers-inspired UI and notification sounds for Kinda Spanish.

## Files

- `daily-reminder.wav`: gentle daily reminder cue
- `streak-reminder.wav`: warm streak reminder
- `streak-warning-soft.wav`: soft warning / lantern dimming cue
- `mission-ready.wav`: new mission ready cue
- `retry-soft.wav`: encouraging retry cue
- `reward-claim.wav`: coins / reward claimed cue
- `success-clear.wav`: useful answer / success cue
- `arcade-unlock.wav`: new arcade station unlocked
- `level-up.wav`: level up / milestone reward cue

These files are served from `/audio/ui/*` in the app.
